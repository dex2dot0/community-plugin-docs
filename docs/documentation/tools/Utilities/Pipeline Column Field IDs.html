Pipeline Column Field IDs

Features​

If you have ever tried adding a column to your pipeline, you'd probably know that the only find option is based on the column name. This is the column name that you set in the reporting database in the Admin Tools.

Wouldn't it be convenient if you could also search by field ID? Of course it would! 😡

That's why the Community Plugin offers this tool. It just makes sense and makes life easier for everyone! ❤️

Setup​

  1. In Encompass
  2. Click on the Settings Menu
  3. Expand the Community Menu
  4. Click on PluginManagement

  1. Select ShowColumnField:

  1. Provide either All Access or more fine grained access by persona or individual user accounts.

Usage​

  1. In Encompass, with a user that has been given permissions to the Grid Search in the steps above:
  2. Select the Pipeline tab
  3. On the pipeline view, right click on any column header and click on Customize Columns...:

  1. After clicking Customize Columns... in the step above, you should now see a new column added that includes the field IDs:

  1. You can now not just see the field IDs but you can also search by field ID as well:
