Navigation Buttons

Featuresâ€‹

Ever want to be able to jump back and forth between input forms like you are used to doing on a web browser? The Community Plugin navigation buttons feature has you covered. ðŸ‘Š

Setupâ€‹

  1. In Encompass
  2. Click on the Settings Menu
  3. Expand the Community Menu
  4. Click on PluginManagement

  1. Select BackAndForward from the Plugin menu
  2. Enable All Access or configure more fine tuned access using either Personas or Users

Usageâ€‹

  1. Once the tool has been enabled, enabled users can now: a. Navigate to an input form in Encompass b. Select the Back button to navigate to the previous form
  2. Additionally, once a user has advanced through multiple forms, if they select the back button to go to the previous form, the forward button will be enabled to allow the user to advance to form they were previously viewing prior to selecting the back button.
